# CLI Reference for ICGL

The ICGL system provides a command-line interface (CLI) for interacting with the system's components.

## CLI Commands

- **`hello`**: Displays a welcome message.
- **`auto [TASK_NAME]`**: Runs an automation task.

## Command Options

- `TASK_NAME`: Optional name of the task to run.

## Usage Examples

```bash
# Display the welcome message
icgl hello

# Run an automation task
icgl auto deploy
```

These examples demonstrate how to use the CLI commands to interact with the ICGL system.
