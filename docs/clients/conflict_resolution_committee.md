# لجنة فض التعارضات الإدارية

## التشكيل

1. مدير التوثيق (DocumentationAgent)
2. مدير السياسات (PolicyAgent)
3. مدير الأرشيف (ArchivistAgent)

## الهدف

حل أي تضارب في الصلاحيات بين الإدارات ورفع التوصيات النهائية لرئيس مجلس الإدارة، مع اقتراح إزالة أو إضافة أو تغيير مهام أو إنشاء وكلاء جدد حسب الحاجة.

## آلية العمل

1. استقبال بلاغات أو رصد تضارب الصلاحيات.
2. اجتماع اللجنة وتحليل الموقف.
3. اقتراح حلول (إزالة/إضافة/تغيير مهام أو وكلاء).
4. رفع التوصية النهائية لرئيس المجلس لاعتمادها.

---

> اللجنة تجتمع عند الحاجة أو عند رصد تضارب جوهري في الصلاحيات أو المهام.
