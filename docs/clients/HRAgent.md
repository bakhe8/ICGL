# HRAgent — مسؤول فحص وتوثيق الوكلاء

## المهمة

- فحص جميع الوكلاء (Agents) الموجودين في النظام.
- تحديد التصنيف المهني لكل وكيل.
- توثيق الواجبات وحدود المسؤوليات لكل وكيل.
- رفع تقرير شامل لرئيس مجلس الإدارة.

## خطوات العمل

1. استكشاف جميع ملفات الوكلاء في src/icgl/agents/.
2. استخراج اسم كل وكيل ووصفه (من docstring أو الكود).
3. تحديد التصنيف المهني (مثال: مراقبة، توثيق، تنفيذ، إلخ).
4. تلخيص الواجبات وحدود المسؤوليات.
5. حفظ النتائج في ملف تقرير (agents_report.md) ورفعها للإدارة.

---

> هذا الوكيل يُنفذ مرة واحدة عند كل تحديث أو إضافة وكيل جديد، ويُعتبر مرجعًا تنظيميًا للهيكل البشري/الآلي في المنظومة.
