# تقرير وكلاء النظام (تلقائي)

| اسم الوكيل           | التصنيف المهني         | الواجبات الرئيسية                                                                 | حدود المسؤوليات                                      |
| ArchitectAgent       | معماري النظام         | تحليل الهيكل، فحص التماسك والفصل، مراجعة حدود النظام                              | لا ينفذ تغييرات مباشرة، يوصي فقط                     |
| ArchivistAgent       | أمين الأرشيف          | حفظ الأرشيف المؤسسي، كشف الفجوات، توليد مسودات السياسات والتقارير                 | لا يعتمد السياسات، يرفع التقارير للإدارة             |
| BuilderAgent         | منفذ بناء             | تحويل القرارات إلى تغييرات ملفات ملموسة، تنفيذ أوامر الإنشاء والتعديل              | يلتزم بالقرارات المعتمدة فقط                        |
| DocumentationAgent   | توثيق                 | تحليل وتحسين جودة التوثيق، كشف النواقص، اقتراح تحسينات                            | لا يكتب الملفات مباشرة، يرفع مقترحات فقط             |
| EngineerAgent        | عمليات/تكامل          | تنفيذ عمليات Git، دمج التغييرات، تنفيذ قرارات الاعتماد                            | لا يقرر السياسات، ينفذ فقط ما تم اعتماده             |
| FailureAgent         | تحليل الفشل            | تحليل سيناريوهات الفشل، تقديم توصيات وقائية                                       | لا ينفذ إصلاحات، يوصي فقط                            |
| MediatorAgent        | وسيط/حل نزاعات         | حل التعارضات بين نتائج الوكلاء، صياغة حلول وسطية أو تصعيد                         | لا يقرر منفردًا، يوصي أو يصعّد                       |
| MonitorAgent         | مراقبة/تحليل          | مراقبة حالة النظام، الإبلاغ عن الحالات الشاذة                                     | لا يتخذ إجراءات تصحيحية، يرفع تقارير فقط             |
| PolicyAgent          | سياسات/حوكمة          | التحقق من الالتزام بالسياسات، مراجعة المقترحات مقابل السياسات                      | لا يعتمد السياسات، يوصي فقط                          |
| SecretaryAgent       | سكرتير سيادي          | ترجمة الأحداث التقنية لتقارير إدارية، بوابة تقارير الرئيس                         | لا يقرر، ينسق ويرفع فقط                              |
| SentinelAgent        | أمن/مخاطر             | تنفيذ قواعد Sentinel، كشف المخاطر والتنبيهات                                      | لا يعتمد القرارات، يوصي أو ينبه فقط                  |
