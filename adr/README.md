# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the ICGL API project. ADRs are used to document important architectural decisions made during the development process.

## Structure

- Each ADR is a markdown file in this directory.
- ADRs are numbered sequentially.
- Each ADR file follows the template provided in `adr/template.md`.

## Usage

1. Copy `adr/template.md` to a new file with the next sequential number.
2. Fill out the template with the relevant information.
3. Commit the new ADR to the repository.
